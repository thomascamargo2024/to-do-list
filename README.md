# 📝 To-Do List

📌 Aplicação simples e moderna de lista de tarefas, desenvolvida para praticar manipulação de DOM e organização de estados com JavaScript.

---

## 🚀 Demonstração

![To-Do List](./imagens/imagem%20to-do%20list.PNG)

---

## 💡 Sobre o projeto

A **To-Do List** permite ao usuário adicionar e remover tarefas de forma simples e intuitiva.

Este projeto foi desenvolvido com foco em:

* Manipulação de elementos no DOM
* Organização de código JavaScript
* Experiência do usuário (UX)

---

## ⚙️ Funcionalidades

* ✅ Adicionar novas tarefas
* 🗑️ Remover tarefas da lista
* 📋 Visualização dinâmica das tarefas
* ⚡ Interface rápida e responsiva

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎨 Interface

O projeto possui um design moderno com:

* 🎨 Gradiente roxo no fundo
* 📦 Cards organizados
* ✨ Interface minimalista e limpa

---

## 🧠 Aprendizados

Com esse projeto, eu aprendi:

* Como manipular o DOM com JavaScript
* Trabalhar com eventos (`click`, `input`)
* Criar elementos dinamicamente
* Melhorar a experiência visual do usuário
